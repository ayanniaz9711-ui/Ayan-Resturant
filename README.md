# 🍽️ Ayan Restaurant

A modern and responsive restaurant website built with HTML, CSS, and JavaScript. The project features an elegant food menu, chef specials, reservation section, and a clean user interface.

## 🌐 Live Demo
https://ayan-resturant.netlify.app/

## ✨ Features

- Modern Restaurant UI
- Fully Responsive Design
- Food Menu
- Featured Dishes
- Reservation Section
- Contact Section
- Smooth Navigation
- Mobile Friendly

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Ayan-Restaurant/
│── index.html
│── style.css
│── script.js
│── images/
└── README.md
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/ayan-resturant.git
```

2. Open `index.html` in your browser.
